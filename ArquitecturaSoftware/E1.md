# Ejercicio 1

*Buscar una aplicación de ejemplo, preferiblemente propia, y deducir qué patrón es el que usa. ¿Qué habría que hacer para evolucionar a un patrón tipo microservicios?*

La aplicación Sistema de Requerimientos Informáticos, es un sistema distribuido con un modelo cliente/servidor tipo REST Client. Utiliza el patrón MVC, el modelo, en Oracle, la vista con extJS y el controlador con AJAX+PHP+JSON.

Para evolucionarla a microservicios habría que desplegar como un servicio independiente, primero lo relacionado al ERP y luego al Sistema de Requerimientos en si, ya que uno alimenta los datos de la empresa y el otro es exclusivo del Sistema de Requerimientos, y solo comparten una fuente de datos comun: el usuario, por lo que no sería inadecuado crear otro servicio que fusione los datos de ambas fuentes.
